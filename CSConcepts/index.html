Upper Schoolroom Cycle 1

Computational Thinking and Algorithms

Unit Content
Computational Logic This unit covers Boolean logic and truth tables, the arithmetic operators, and boolean operators.
Algorithms Students should be able to plan a simple program with success criteria, test data and simple algorithms. Students will learn about flowcharts, and binary addtion and conversion of 4-bit numbers
Programming Techniques Students will learn about: Variables, Input, Output, Storage of data, Programming Constructs, Operators, and Comments
Data Representation Students will learn, Units of computer memory, Data structures and compression, and text, sound, and images.

Upper Schoolroom Cycle 2

Computer Systems

Unit Content
Computer Hardware Compinents of a computer, internal components of a computer and their function (Motherboard, CPU, RAM, BIOS, and Hard Disks), and Peripherals and their function (Camera, Keyboard, Microphones, Monitor, Mouse, Scanner, Headphones, Speakers, Printer)
Computer Software Operating Systems, System Software, Utility Software (Computer Security, Disk management, System Maintenance), and types of application software (Image Processing, Word Processing, Spreadsheets, Web Browsers, Presentation Software, Database, IDEs)
Computer Memory and Storage Primary Storage (Main Memory) and Secondary Storage (Magnetic, Optical, Flash[Solid State]), appropriate uses of secondary storage
Moral, Legal, Cultural, and Environmental Concerns Moral issues, Legal issues (Data Protection, Computer Misuse, and Copyright), Environmental issues, open source and proprietary software

Upper Schoolroom Cycle 3

Programming Project

Candidates take on one of three programming projects that are then marked against criteria. Students need to show that they can analyse, design, develop, test, and evaluate their solutions.

About the Cycle

The cycle covers two units of Vector graphics and Computing Systems.

In vector graphics students will be shown how to use computing tools to create sophisticated illustrions such as logos, icons, and sprites

The computer systems unit will show students the layers of computing systems from how programs are made to the hardware the PC uses

Essential Outcomes

  • Use Drawing tools to draw vector images
  • Explain differences between vector and bitmap images
  • Create a project plan and evalue the plan to the purpose
  • Understand the hardware ans software in a computer system
  • Undersatand simple boolean logic
  • Understand how instructions are stored and executed

Wider Outcomes

  • The link between computer systems and our own brains
  • How adverts seen around link to different vector tools

About the Cycle

The cycle covers two units of clear messaging in digital media, and from semephores to the internet

Clear messaging in digital media is the unit that encompasses e-safety and the orientation of common software on the system.

From semaphores to the internet shows the basics of networks and how the internet works different hardware is discussed along with the World Wide Web.

Essential Outcomes

  • Using a range of tools to create media
  • Communicate ideas effectively and efficently
  • Understand branding and why it needs to be consistant
  • Identify common networking hardware
  • Explain the difference between the internet and the world wide web
  • Understand and identify different protocols

Wider Outcomes

  • Understand how data is collected and monitored
  • Understand why companies need to use and monitor data
  • Understand that what I do online is permenant
