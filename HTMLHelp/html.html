The basic set up of a HTML file

HTML provides the structure of a webpage, on the right you can see the code of a basic webpage. We will go through the different tags that you need to know below the example.

The tags you need to know are outlined in the specification which you can find here

A-Level Spec
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>HTML Help</title>
  </head>
  <body>
    <h1>Welcome to HTML</h1>
    <p>This is all about HTML</p>
  </body>
</html>

Headings

Headings are really easy to add to any HTML document. You can use any of the tags that are listed to the left. These would go into the body tag of your HTML file like in the example at the top of the page. H1 is the biggest all the way down to H6 which is the smallest.

Paragraphs

Paragraphs easily add text to a webpage. This is usually small text that is formatted as a paragraph. It is the basic building block of any webpage. To add them to your file you use the p tag as shown on the right

<p>This is a paragraph.</p>
<ol>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Ordered and Unordered lists

Ordered lists are lists that are numbered and use the ol tag e.g.

  1. Item1
  2. Item2
  3. Item3
These could be used as rankings for example within a webpage

Unordered lists use bullet points instead. This uses the ul tag e.g.

  • Item1
  • Item2
  • Item3

The li tag means listed item. It is an item that appears within the list

Images

Images can be quite difficult, this is because they are self closing tags so they don't have a pair. They also require two attributes: the source or src which is where the image is located and alt which means alternative test for the image. Have a look at the example and try spotting the source and alternate text.

<img src="balloons.jpg" alt="a picture of baloons"/>
<p>To go to Google <a href="www.google.com" target="_blank">click here</a></p>

Links

A link is used with an a tag. These anchor other websites/pages to your page.

You need to add an attribute of href to say where you want the link to go. If you want the link to open in a new tag then we also add the attribute for target. Between the opening closing tags we add the text that will be clickable.

Div

A div means a division of a webpage. You can use divs to group elements together that you might then style as a whole later on.

<div>
  <!-- Other elements go here -->
</div>
<form>
  <label for="fruit">Favourite fruit:</label><br>
  <input type="text" id="fruit" name="fruit"><br><br>
  <input type="submit" value="Submit">
</form>
