A birthday Card

Create a birthday card for a relative that has intractive elements. Your birthday card should:

  • Say happy birthday on it
  • Have different sprites moving around
  • Use a sprite to create a streamer
  • Use sound to make the card sound like a party

Challenge

Ask the user for the birthday person's name and add it to the happy birthday message

Space Invaders

It is time to create a game of space invaders. This can be quite a tricky project so use the requirements below to create an amazing game.

  • Create a sprite of a spaceship for the user to control
  • Create sprite of the alien invaders
  • Allow the user to move the spaceship left and right with the arrow keys
  • Write code for the alients to move left, right, and down so they get closer and closer to the player
  • Create a sprite that can be a laser that the spaceship can shoot
  • Create code for if the laser touches an invader then it hides
  • Create a you win screen for if all the aliens are defeated
  • Create a you lose screen if the aliens touch the spaceship
