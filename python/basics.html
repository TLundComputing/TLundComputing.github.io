Printing, Variables, and Inputs

Printing

Printing is a very simple but useful feature of any programming language. Python makes this really easy for new and experenced programmers. Have a look at the example to the right (or below on mobile device).

Printing Code Example

print("Hello World")
print("This is super easy to do!")

Variables

Variables are used to store information while a program is running.

Variables can be seen a bit like a cardboard box that you assign a label. for example in the code snippet to the right (or below on mobile), you can see that there is a variable that is called name, you can see that I have changed the value of name a couple of times. Output is shown via the hash character (#), when the program is run you will see it differently.

You can use these to be able to temporaily store data that is important to your program. What variabels would we use in a quiz game?

Variables Code Example

name = "Tom Lund"
print("Your name is", name)
#Your name is Tom Lund

name = "Mickey Mouse"
print("Your name is", name)
#Your name is Mickey Mouse

Inputs

Inputting data makes your program more flexible and less static. To do this you need to use variables like in the section earlier, the keyword input is then used to request information from the person using your program (user). We shall say our user will enter Donald Duck, again I shall use console output with the # symbol.

Input Code Example

name = input("Enter your name: ")
print("Your name is", name)
#Your name is Donald Duck
