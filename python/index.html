Getting Started

How to use this tutorial

This tutorial is split into sections on the left. If you are viewing on a small screen or a mobile device, then please click the "burger" icon at the top of the screen to show the sub-menu. This tutorial is going to go over some basic skills that cover the OCR GCSE and A-Level specification. There are other indepth tutorials out there and I recommend that you find one that is right for you. Code examples will be shown to the side of any descriptive text as shown below (If you are on a small or mobile device it will stack underneath).

Python is a very versatile language that has been used in many different applications. Even though Python is a general purpose language, it can be used in embedded systems that perform a single purpose.

Python is very easy to install but you can use a web environment if you do not want to install the language on your computer.
