Introduction to CSS

CSS stands for cascading stylesheet. You use this to add style to your website. To do this you can do it inline or in a seperate CSS file using the <link> tag. For the A-Level qualification you need to know the different identifiers (covered later in this document) and the following properties that you can change (click on the properties to see how they work):

Structure of a CSS statement

With CSS you need to use a selector then change that element with different attributes that are in curly brackets. Attributes and properties are seperated with a colon and ended with a semi colon.

body {
  font-family: Arial, sans-serif;
  font-size: 12pt;
}

The above changes the font of any text within the body tag to arial with a font size of 12.

Selectors are also used. to selefct what you want to change the style of. If you want to select all the instances of a tag, then you just use the tag name. If you want a specific element then you can use an ID, a selector with an ID uses a # infront of the selector name. An ID can only be assigned to one alement of your html file. If you want some elements selected but not all then you can select a class, classes can be added to any element and reused through the html file. Class selectors use a . at the beginning of the name.

Changing the background colour

The background colour can be changed by stating the name of the element, the class of the element, or the id of the element. To give a background colour you can use hex values in the for red blue green, or you can state a names colour. Look at the code to the right for an example of changing the background colour of the body tag.

body {
  backround-color: #428af5;
}

Borders

Borders go around elelments with solid, dashed, or dotted lines. You can change the the colour and thickness of the borders for each element. you can do this seperatly or you can do it all at once. Look at the example and have a look at how the code changes for giving two images the same border.

#potrait {
  border: 5px solid red;
}


#landscape {
  border-style: solid;
  border-width: 5px;
  border-color: red;
}

Font

Changing the font comes in a couple of steps looking at the font family and the font size. You need to use a selector to choose what text you want to change. The example shows how to change the font to comic sans for paragraphs.

p {
  font-family: "comic sans";
  font-size: 12pt;
}

Dimentions

You can change the dimentions of different divisions, images, and other elements. To do this you add a width and a height attribute within your css. Look at the square below. This is a red square with the id red. I have used CSS to make it 100px by 100px

#red {
  width: 100px;
  height: 100px;
  background-color: red;
}
